# EMBL-EBI Google Summer of Code Projects

Welcome to the **EMBL-EBI Google Summer of Code (GSoC)** project repository 🎉

The European Bioinformatics Institute (EMBL-EBI) is a global leader in biological data, developing open data resources and open-source software used by researchers worldwide. 

This repository contains project code, documentation, and resources for the EMBL-EBI GSoC 2026 participation.

## About Google Summer of Code

{Google Summer of Code (GSoC)](https://summerofcode.withgoogle.com/) is a global programme that brings new contributors into open-source software development.

At EMBL-EBI, GSoC projects typically focus on:

- Bioinformatics tools and data infrastructure

- Open-source software for life sciences

- Data visualisation and analytics

- Machine learning applications in biology

- Improving usability and accessibility of scientific resources
